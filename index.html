Scaffold

Scaffold is a ultra flexible and leightweight sass based grid system.

It's designed to give you flexible foundations to build what ever you want easily.

Check out the repository on github.

Install and Build

A prebuilt version of Scaffold with default settings can be found in the build folder, you can create a custom build from the supplied SASS file.

Each featured is seperated into its own file and can be added/removed from the main scaffold.sass. Variables can be adjusted from the vars.sass file.

To build a custom version of Scaffold you can use the original SASS file along with the provided gulpfile to build the css file. You can adjust the amount of columns, breakpoints, gutters and more.

Custom Build

You can create a custom build using the following steps

  • Install node and npm
  • Install gulp (globally) npm install -g gulp
  • Install dependancies npm install
  • Build file gulp build

The Grid

Scaffold is all about the grid, the sass file can genreate any amount of grids with varying breakpoints and columns with adjustable gutters.

Each breakpoint added will create a new named grid set that will respond to width between its width and the next width up.

Grid Sizes

The Grid is built from the $breakpoints variable that have been set up in the vars.sass folder.

Adding a new breakpoint and grid is as easy as adding a new key name and matching pixel value as follows:

$breakpoints: (small: 320px, medium: 768px, large: 1024px, extra-large: 1280px)

We can then define columns per breakpoint as follows:

  • .col-8-large
  • .col-8-medium
  • .col-8-small

If the screensize drops below our specified breakpoint and there are no classes added to support the next breakpoint the column will default to fullwidth.

Grid Ranges

Each grid size only is only applicable within its breakpoint, if we want to keep the same column size over a range of sizes we can append -up to a class.

To create a 4 column div for medium and large screens we add the class .col-4-medium-up.

  • Default breakpoints:
  • small - 320px
  • medium - 767px
  • large - 1024px
